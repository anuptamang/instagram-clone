# Instagram Clone App
## Demo: https://instagram-clone-725f8.web.app

## Technologies Used:
- React
- Hooks
- React Router (react-router-dom)
- Context
- Firebase DB, Auth, Hosting, Storage
- Tailwind CSS
- Material Icons
- Axios

## Features available so far:
- User Signup / Sign in
- User Email Verification
- User Password Reset
- Post Create (image/video posting)
- Post Comment
- Emoji Comment
- Post Like
- Single Post
- Basic Page Routes
